# BENCH_v.2

## Overview
* **Paper:** [Assessing the macroeconomic impacts of individual behavioral changes on carbon emissions](https://link.springer.com/article/10.1007/s10584-019-02566-8) (Climatic Change, 2020)
* **Authors:** Leila Niamir, Gregor Kiesewetter, Fabian Wagner, Wolfgang Schöpp, Tatiana Filatova, Alexey Voinov, Hans Bressers
* **Code DOI:** [![DOI](https://img.shields.io/badge/DOI-10.5281%2Fzenodo.20847236-blue)](https://doi.org/10.5281/zenodo.20847236)


## Technical Details
* **Language:** NetLogo 6.x
* **Main File:** `BENCH_v.02.nlogo`
* **Extensions Used:** `csv` (for loading data)

---

## Quick Start

1. Open `BENCH_v.02.nlogo` in NetLogo (v6.0 or higher).
2. Click **Setup** to initialize the environment and load data.
3. Click **Go** to start the simulation.
